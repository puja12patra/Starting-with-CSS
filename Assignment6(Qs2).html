BEM stands for Block, Element, and Modifier. It's a CSS naming convention for writing cleaner and more readable CSS classes. It communicates component structure It sets a consistent low-level of specificity for styling selectors

I'm a light card

The outermost parent element of the component is defined as the block.
Inside of the component may be one or more children called elements.
Flags on blocks or elements. Use them to change appearance, behavior, or state.

Here, classname "card" is a Block. "card--dark" and "card--light" is a Modifier. "card__image","card__body","card__title" are Elements which follows the BEM naming convension.
